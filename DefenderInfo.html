Defender!

About

This is a 2D game where the player has to protect a castle from flyiong monsters. This game was developed to practice 2D physics on the elements of the game
Currently, you can play it on a PC or mobile device through a web browser. However, there are plans to bring it to the Google Play Store.

Project info

Role: Game programmer & designer
Team Size: 1
Duration: 2 weeks
Engine: Unity (C#)

Development

My responsabilities during this project were:
- Design and development of the game system.
- Implemented the audio for the game
- Creation of a manager responsible for switching views between screens, as the game operates within a single scene.
- Developed a dificulty system to improve the enemies according to score
- Developed a simple 2D physics for the game elements (movement and behaviors related)
- developed a "drag & drop" system for the main mechanic of the game and for the player's feedback
