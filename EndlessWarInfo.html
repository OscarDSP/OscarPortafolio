Endless War

About

This is a 3D survival game where the player must enter a forest to survive waves of zombies where each wave is more difficult than the previous one.

Project info

Role: Game programmer & designer
Team Size: 1
Duration: 3 weeks
Engine: Unity (C#)

Development

My responsabilities during this project were:
- Design and development of the game system.
- Implemented the audio for the game
- Implemented the animations for the player and the enemies
- Developed a wave system capable of showing remaining enemies. It is also capable of improving the zombies for the next wave.
- Developed an ammunition and weapon system which currently contains 2 weapons (main and secondary).
