Procedural Terrain

About

This experiment was made with the objective to be implemented in one of my games called "New World" which consist in the creation of NPCs so they can survive to the environment and with this, I want to experiment with more complex AI systems.
With the creation of this new terrain, which uses "Noise" and other components to modify it, it is expected not only a visual improvement but also a better and more complex behavior for the agents in the environment and terrain

Project info

Role: Game programmer & designer
Team Size: 1
Duration: 4 months
Engine: Unity (C#)

Development

My responsabilities during this project were:
- Developed the algorithm that can create "Noise" to create the first layer of terrain. Once finished, it will be passed through a second layer which adds octaved to smooth the terrain and make it look more natural that just with "Noise".
Once with the terrain generated, it is necessary to color it or apply the visuals so that it looks good when playing it. For that, the values from 0 to 1 are taken and colors are applied to the appropriate values and with this we would have an algorithm that generates beach, field, mountain and snow and can easily be expanded to generate more biomes.
