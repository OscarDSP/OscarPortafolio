Mewpocalyps

About

Mewpocalyps was developed for the Unreal championship organized by EpicGames, it was made by 4 programmers and 4 designers. and the main objective was to achieve a survival game in which cats were the main game mechanic.

Project info

Role: Gameplay programmer
Team Size: 8
Duration: 3 Months
Engine: Unreal Engine (C++)

Development

As a gameplay programmer, my responsabilities during this project were:
- Developed the movement controller which consisted of a simple Top-Down movement using the "WASD"
- Developed the camera controller which consisted of a third-person view that the player can rotate 90 degrees in any horizontal direction
- Developed the core mechanic of the game, this was a laser that points at the mouse and it has 2 states. When the laser is off and if the player uses the left mouse click, it turns on, if the player hit a zombie when the laser is on, then a cat will go attack it.
- Developed a system for the character and NPCs animations so it can face the right directions depending on the camera angle. - Implemented the audio for all the sound effects and music in the game.
- Developed secondary mechanics.
