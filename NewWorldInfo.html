New World

About

This game was made to experiment with advanced AI where players must create NPCs and they will have some behaviours in order to survive the environment.

Project info

Role: Game programmer & designer
Team Size: 1
Duration: started at December, 2022 - currently under development
Engine: Unity (C#)

Development

My responsabilities during this project were:
- Designed and developed the game system
- Implemented the audio for the game
- Developeded an algorithm to generate terrain procedurally. This algorithm uses cellular automata to generate a first layer of terrain, then it goes through a second layer to make it look more natural.
- Developed the AI for the NPCs which have an advanced perception system and fuzzy logic for the decision making depending on what the NPC needs
- Developed a netwrok which takes the information of the current generation of NPCs in order to create a new generation, it uses the strongest stats to improve the normal stats of the new generation.
