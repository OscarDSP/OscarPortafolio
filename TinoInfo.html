Tino

About

Tino is a plataformer where the players must go through small levels using the different mechanics available for the player and each level is harder than the last one

Project info

Role: Game programmer & designer
Team Size: 1
Duration: 3 weeks
Engine: Unity (C#)

Development

My responsabilities during this project were:
- Design and development of the game system.
- Implemented the audio for the game
- Developed a side view controller for the player
- Developed core and secondary mechanics such as a dash and a kockback attack
- Developed a level selector
- Developed basic AI for the enemies with a perception system
- Implemented the animations for the player and the enemies
